# Assets Folder

This folder contains application assets for Cimian Software Center.

## Required Assets

The following icons should be added for MSIX packaging:

- `StoreLogo.png` - 50x50 store logo
- `Square150x150Logo.png` - 150x150 tile  
- `Square44x44Logo.png` - 44x44 app list icon
- `Wide310x150Logo.png` - 310x150 wide tile
- `SplashScreen.png` - 620x300 splash screen
- `LockScreenLogo.png` - 24x24 lock screen badge

## Icon Generation

You can generate these from a single 1024x1024 source icon using tools like:
- Visual Studio Asset Generator
- Figma with export slices
- ImageMagick command line

## Placeholder

Until proper icons are created, the app will use Windows default icons.
